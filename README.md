# Vitalia Landing Page

Official landing page for **Vitalia**, a comprehensive healthcare management platform developed by **KinetiaLabs**.

## Description

Vitalia is a platform designed to simplify the management of clinics and medical centers. This landing page presents the main features, available plans, and contact information.

## Features

- Responsive and modern design
- Multilingual support (English / Spanish)
- Sections: Features, Clients, Plans, Product, Contact, About, Team
- Contact form

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Custom translation system

## Usage

1. Clone the repository
2. Start a local server (required for translations to work):
   ```bash
   npx serve .
   ```
3. Open `http://localhost:3000` in your browser
4. Use the language selector to switch between English and Spanish
